# Meta Video Generator

A modern browser extension built with [WXT](https://wxt.dev) and React.

## Features

- Built with WXT (Next-gen framework for building browser extensions)
- React 19 for UI components
- TypeScript support

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Start development server for Firefox
npm run dev:firefox
```

## Build

```bash
# Build for production
npm run build

# Build for Firefox
npm run build:firefox

# Create ZIP archive
npm run zip
```
